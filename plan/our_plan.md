# Portfolio Website Plan: Aura Gold (DIY Necklaces)

## Objective
Create a modern, DIY necklace portfolio website for "Aura Gold" hosted on GitHub Pages, featuring a pink pastel theme, JetBrains Mono font, a hero section, 3 project showcases, and social/e-commerce links.

## Key Files & Structure
- index.html: Main structure and hero section.
- style.css: Pink pastel theme, responsive layout, JetBrains Mono font integration.
- script.js: Basic interactivity (e.g., project filtering or simple animations).
- assets/: Folder for product images and icons.

## Implementation Steps

### Phase 1: Setup & Design
1. Initialize a new GitHub repository named aura-gold.github.io (or a project repo).
2. Set up the folder structure (index.html, css/style.css, js/script.js, assets/).
3. Define the design system:
    - Color Palette: Soft pinks (#FFD1DC), light pastels (#FFF0F5), and a clean neutral for contrast.
    - Typography: Import JetBrains Mono via Google Fonts.

### Phase 2: Content Development
1. Hero Section: Add branding "Aura Gold" with a welcoming statement.
2. Projects Section: Create a 3-column grid to showcase 3 necklace collections with images, descriptions, and "View" links.
3. Contact/Social Section: Integrate a styled list or button group linked to your Linktree (E-commerce and Social).

### Phase 3: Deployment
1. Commit and push files to GitHub.
2. Enable GitHub Pages in the repository settings (Settings > Pages > Deploy from branch).
3. Verify the site is live at https://[your-username].github.io/aura-gold/.

## Verification & Testing
- Check mobile responsiveness (CSS Media Queries).
- Validate links to Linktree.
- Confirm fonts and color theme load correctly.
