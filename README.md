# Aura Gold Portfolio

Welcome to the Aura Gold Portfolio project! This is a personal portfolio website designed to showcase professional work and creative projects.

## Project Structure

- `index.html`: The main entry point for the portfolio website.
- `css/`: Contains the stylesheet (`style.css`) for the website's design.
- `js/`: Contains the JavaScript file (`script.js`) for interactivity.
- `plan/`: Includes the project planning documentation (`our_plan.md`).

## Features

- Responsive design.
- Interactive elements.
- Professional layout.
